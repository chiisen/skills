# 範例引導 (Examples)

以下是「一句話生圖」的轉化範例，AI 應遵循此邏輯進行腦補：

## 範例 1：技術開發情境
**用戶輸入：** 波弟在寫 PHP Laravel 時遇到 Bug 生氣了。
**AI 腦補分鏡 (B4)：**
1. 特寫：波弟盯著螢幕，爪子快速敲鍵盤。
2. 畫面：螢幕顯示紅色的 Error Exception。
3. 中景：波弟尾巴炸毛，眼神變得凌厲。
4. 遠景：波弟跳上辦公桌大吼「喵！（這 Code 誰寫的！）」。
5. 動態：波弟使出「連環貓爪」瘋狂抓螢幕。
6. 效果：螢幕碎裂，狀聲詞「鏘！」。
7. 切換：波弟癱坐在鍵盤上吸貓薄荷冷靜。
8. 結尾：螢幕奇蹟般地恢復正常，波弟露出狡黠的笑容。

## 範例 2：日常生活情境
**用戶輸入：** 波弟發現了一罐傳說中的極品雞肉罐頭。
**AI 腦補分鏡 (B4)：**
1. 遠景：波弟在廚房高處俯視。
2. 特寫：發現櫃子深處發光的罐頭。
3. 動態：優雅地跳下，空中翻轉。
4. 效果：降落在罐頭前，雙眼發光（狀聲詞：閃亮！）。
5. 動作：努力用貓爪摳開罐頭蓋。
6. 近景：罐頭打開，肉香四溢（狀聲詞：嚼嚼）。
7. 表情：波弟吃到流淚的幸福臉。
8. 結尾：波弟躺在罐頭旁摸肚子，背景滿是小花。
